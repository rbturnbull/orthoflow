Usage

After installing, you need to create an input file for phyloflow to use. The default name for this file is input_sources.csv but this can be changed in the configuration file.

The file looks something like this:

It needs the columns file, taxon_string, data_type and translation_table. The file column is the path to the file relative to the working directory. The taxon_string, is the name of the taxon in the file. The data_type column must be CDS or GenBank, gb, or gbk (case insensitive). The translation_table is integer id of the translation table to translated coding sequences to amino acid sequences using BioKIT. See the list of genetic codes published by the NCBI.
